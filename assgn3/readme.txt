For compiling the Semaphore program :
g++ prod_cons-sems-CS17BTECH11037.cpp -pthread -std=c++11

For compiling the Mutex-Lock program :
g++ prod_cons-locks-CS17BTECH11037.cpp -pthread -std=c++11 

For running the program:
./a.out

Input will be taken from 'inp-params.txt' and output will be generated in "<program>-Log.txt" and
